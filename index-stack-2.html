Every Layout - The Stack, Grow My and Natural Me

Usage of css class tw-stack-byflex-grow-my on the usage-exampleReset , cascading by default to all children which do not exempt - like the grow-me-element-1Reset and grow-me-element-2Reset - to make them grow to the remaining height.

Usage of the css class tw-stack-byflex-natural-me on the natural-me-element-1Reset and natural-me-element-2Reset , as an exempt override to tw-stack-byflex-grow-my, coming from the usage-exampleReset . The exempt override reacts opposite, in that it will only consume its's intrinsic, natural space.

This element will be replaced by any element iff a root element of id usage-example exists and having an attribute of for-code-example. Then any node having an attribute of for-code-example is selected, formatted and highlighted here.

The stack's assigned space is colored for demonstration.
