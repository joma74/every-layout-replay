Example Three

Usage of css class tw-stack-2 on the container-element-1, container-element-2 and container-element-3 cascading the spacing between the stack's elements in a vertical rhythm based on the Tailwind CSS spacing scale No. 2 (2 * 0,25rem = 0,5rem).

Usage of css class tw-stack-justify-start on the container-example to soak every element up, but css class tw-stack-springafter-2 on the same container-example advises the margin after the container-element-2 to act like a spring, pushing the container-element-3 down to the bottom.

The stack's assigned space is colored for demonstration.
