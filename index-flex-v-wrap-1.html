Flex, Column Wrap, Spec Bug Workaround, Writing Mode. Not Working

This serie shows a flexbox of vertical orientation, where wrapping is turned on. It is a follow up to the "Sidebar, Horizontal Then On Wrap Vertical, Spec Bug" serie , where the sidebar pattern was ruled out because it is said that a spec bug prevents proper implementation in all browsers.

As been said in the forums, an application of writing-mode: vertical-lr on the container and a writing-mode: horizontal-tb on the flex item should be a workaround. But it is not.

For reason of not working, no further description is provided. See the headings below on the examples and code examples to see what they are reaching for.
