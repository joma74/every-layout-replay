Every Layout - The Center, Horizontal, Just Margins

A css class of tw-center-h-4 centers the content, shrinking it with a max-width of max-content and hence leaving application for the margins to the left and right, in effect centering their content. The paddings to the left and right of the tw-center-h-4 is based on the Tailwind CSS spacing scale No. 4 (4 * 0,25rem = 1rem). The paddings to the left and right remain, because the box-sizing value of all tw-center-h-* is content-box.

The first example, centerReset shows an application of tw-center-h-4, hence the tw-center-h-4 element shrinks to it's instrinsic content and leaving application for the margins to the left and right, in effect centering their content.

The second example, minimal-containerReset , additionally applies a max-width hint of max-content, this time to the container of the element defining the tw-center-h-4. Hence the p does not expand, leaving no application for the margins to the left and right. Still, the paddings to the left and right remain.

The third example, sufficientReset , at last, just shows the minimal sufficent application of tw-center-h-4.

This element of id code-example will be replaced by code iff a root element of id usage-example exists and having an attribute of for-code-example. Then any node having an attribute of for-code-example is selected, formatted and highlighted here.

In the first and second example, the element defining the tw-center-h-4 is colored with tw-bg-purple-400. It's container is colored with tw-bg-yellow-100, while it's content element is colored with tw-bg-blue-500. To show these colored effects, the DOM of the examples is contrived into multiple HTML elements. In practice, just a single element with tw-center-h-4 would be sufficient - this is what the third example shows.
