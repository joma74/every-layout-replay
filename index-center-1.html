Every Layout - The Center, Just Margins

A css class of tw-center-4 centers the content, padding the element based on the Tailwind CSS spacing scale No. 4 (4 * 0,25rem = 1rem). All of the coming examples make use of tw-center-4. Also, in all of the coming examples the paddings to the left and right remain, because the box-sizing value of all tw-center-* is content-box.

The first example, shrinked-centerReset , additionally applies a max-width hint of max-content to the element defining the tw-center-4. Hence the tw-center-4 element shrinks to it's content, leaving application for the margins to the left and right, in effect centering their content.

The second example, shrinked-contentReset , does not apply any max-width hint. Hence the p block always expands to it's container's left and right edges, leaving no application for the margins to the left and right.

The third example, shrinked-containerReset , additionally applies a max-width hint of max-content, this time to the container of the element defining the tw-center-4. Hence the p does not expand, leaving no application for the margins to the left and right.

The fourth example, shrinked-nothingReset , does not apply any max-width hint. Hence the p block always expands to it's container's left and right edges, leaving no application for the margins to the left and right.

This element will be replaced by any element iff a root element of id usage-example exists and having an attribute of for-code-example. Then any node having an attribute of for-code-example is selected, formatted and highlighted here.

The element defining the tw-center-4 is colored with tw-bg-purple-400. It's container is colored with tw-bg-yellow-100, while it's content element is colored with tw-bg-blue-500. To show these colored effects, the DOM of the examples is contrived into multiple HTML elements, applying tw-bg-clip-content where appropriate. In practice, just a single element with tw-center-4 would be sufficient.
