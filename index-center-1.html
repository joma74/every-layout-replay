Every Layout - The Center, Just Margins

A css class of tw-center-4 centers the content, padding the element based on the Tailwind CSS spacing scale No. 4 (4 * 0,25rem = 1rem). Both of the two next examples make use of tw-center-4.

The first example shrinked-to-contentReset additionally applies a max-width hint of max-content. Hence the p block shrinks to it's content, leaving application for the margins to the left and right, in effect centering their content. The paddings to the left and right remain, because the box-sizing value of all tw-center-* is content-box.

The second example not-shrinked-to-contentReset does not apply any max-width hint. Hence the p block always expands to it's parent's left and right edges, leaving no application for the margins to the left and right. The paddings to the left and right remain, because the box-sizing value of all tw-center-* is content-box.

This element will be replaced by any element iff a root element of id usage-example exists and having an attribute of for-code-example. Then any node having an attribute of for-code-example is selected, formatted and highlighted here.

The elements' padding is colored with tw-bg-purple-400, while the elements' margin is colored with tw-bg-yellow-100. The elements' content is colored with tw-bg-blue-500. To show these colored effects, the DOM of the examples is contrived into multiple HTML elements. In practice, just a single element with tw-center-4 would be sufficient.
