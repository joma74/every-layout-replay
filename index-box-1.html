Usage of css class tw-box-2 on box-1Reset , box-2Reset and box-3Reset , padding the element based on the Tailwind CSS spacing scale No. 2 (2 * 0,25rem = 0,5rem).

By forcing inheritance, you can change the color — along with the background-color — in one place: on the usage-exampleReset itself, here by applying a css class of tw-text-pink-400 and a css class of tw-bg-blue-500.

This element will be replaced by any element iff a root element of id usage-example exists and having an attribute of for-code-example. Then any node having an attribute of for-code-example is selected, formatted and highlighted here.

To demonstrate the difference of the padding from the content by colors of each tw-box-2, the content is clipped by usage of the css class tw-bg-clip-content. Hence the box's assigned padding is colored for demonstration.
