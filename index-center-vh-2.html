Every Layout - The Center, Vertical And Horizontal, By Flex, By Margin

This serie shows how to horizontal and vertical center elements by applying margins to a flex box and distribute them uniform.

1-h-center-vhReset shows an application of tw-flex-h tw-center-vh-bymar-myon the container of a button, which is oriented in a direction of column.

2-v-center-vhReset shows an application of tw-flex-h tw-center-vh-bymar-myon the container of a button, which is oriented in a direction of row.

3-h-center-vh-uniform-fromconReset shows an application of tw-flex-h tw-center-vh-bymar-myon the container of two buttons, which is oriented in a direction of row. The uniform distribution of the buttons along the v line is an instrinsic effect of applying tw-center-vh-bymar-my on the container. 3-h-center-vh-uniform-fromelmReset shows an application of tw-flex-hon the container of two buttons, which is oriented in a direction of row. The uniform distribution of the buttons along the v line is an instrinsic effect of applying tw-center-vh on each button element.

4-v-center-vh-uniform-fromconReset shows an application of tw-flex-v tw-center-vh-bymar-myon the container of two buttons, which is oriented in a direction of column. The uniform distribution of the buttons along the h line is an instrinsic effect of applying tw-center-vh-bymar-my on the container. 4-v-center-vh-uniform-fromelmReset shows an application of tw-flex-von the container of two buttons, which is oriented in a direction of column. The uniform distribution of the buttons along the h line is an instrinsic effect of applying tw-center-vh on each button element.

This element will be replaced by any element iff a root element of id usage-example exists and having an attribute of for-code-example. Then any node having an attribute of for-code-example is selected, formatted and highlighted here.

A class of tw-flex-1 on each example is applied only to make those, being each a flex-item themselves, the same height. If there is either negative or postive space to be distributed, then regardless of their individual natural size, after distributing the space proportional to the specified flex factor, the one with the maximal min-size dictates the effective size of the others.
