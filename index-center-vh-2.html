Every Layout - The Center, Vertical And Horizontal, By Margins

This serie shows how to horizontal and vertical center elements by applying margins which are contained in a flex box.

1-center-v-h-flex-colReset shows an application of tw-center-v-4 and tw-center-h-4 on a button, which is contained in a flex box oriented in a direction of column.

2-center-v-h-flex-rowReset shows an application of tw-center-v-4 and tw-center-h-4 on a button, which is contained in a flex box oriented in a direction of row.

3-center-v-h-flex-colReset shows an application of tw-center-v-4 and tw-center-h-4 on two buttons, which are contained in a flex box oriented in a direction of col. Note that beacuse of the direction of col on the flex box container, the buttons are arranged row wise.

4-center-v-h-flex-rowReset shows an application of tw-center-v-4 and tw-center-h-4 on two buttons, which are contained in a flex box oriented in a direction of row. Note that beacuse of the direction of row on the flex box container, the buttons are arranged column wise.

This element will be replaced by any element iff a root element of id usage-example exists and having an attribute of for-code-example. Then any node having an attribute of for-code-example is selected, formatted and highlighted here.

A class of tw-flex-1 on each step-* container is applied only to make those, being each a flex-item themselves, the same height. If there is either negative or postive space to be distributed, then regardless of their individual natural size, the one with the maximal min-size dictates the effective size of the other.
